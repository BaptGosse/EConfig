Bienvenue sur EasyConfig, allias EConfig ;)

EConfig est un moyen rapide pour configurer votre serveur minecraft avec des interfaces incroyable et très simple d'utilisation, une documentation hors norme et des addons pour vous garantir une grande compatibilité avec d'autres plugins. N'hésitez plus, passez à EConfig...

La page de téléchargement vous permet de télécharger notre plugin ainsi que notre mod spécial qui vous simplifiera la configuration de votre serveur. Merci à PixelDeEcran pour le code qui lui a demander presque 200 heures de travail
